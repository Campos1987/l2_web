11. Deploy e Ambientes

Recomendado: Vercel ou VPS (Docker).

Produção:
1. Configurar variáveis de ambiente no painel do host.
2. Buildar o projeto: `npm run build`.
3. Iniciar servidor: `npm start`.

Se hospedar em VPS junto com o L2Server:
- Use PM2 para gerenciar o processo Node.js.
- Configure Nginx como Proxy Reverso para a porta 3000.
- Certifique-se que o banco de dados permite conexões locais ou configure o firewall.
