5. Estrutura de Pastas (Refatorado - Tech Lead Standard)

src/
 ├─ app/                 -> Camada de Rotas (Next.js App Router)
 │   └─ register/        -> Página de registro
 ├─ modules/             -> Lógica de Negócio (Domain Driven)
 │   ├─ auth/            -> Módulo de Autenticação (Actions, Schemas)
 │   ├─ register/        -> Módulo de Registro
 │   └─ query-service/   -> Serviços de Leitura (CQRS-lite)
 ├─ ui/                  -> Design System e Componentes Genéricos
 │   ├─ Button/          -> Componentes atômicos
 │   ├─ Header/          -> Header e subcomponentes
 │   ├─ Footer/          -> Footer
 │   ├─ Carousel/        -> Carousel customizado
 │   └─ Logo/            -> Logo da aplicação
 ├─ infra/               -> Infraestrutura e Conexões
 │   └─ db.ts            -> Instância do Prisma Client
 ├─ lib/                 -> Utilitários Gerais
 │   └─ validation/      -> Helpers de validação
 └─ styles/              -> CSS Global

prisma/
 └─ schema.prisma        -> Schema do banco
